OverView.

이 웹사이트는 이전에 진행했던 WeddingLery 프로젝트의 소스를 활용하여 JSON 데이터를 파싱하고, 화면에 동적으로 표시하도록 개발했습니다. 로딩 과정에서는 스켈레톤 UI를 먼저 화면에 표시해 사용자에게 로딩 상태를 시각적으로 알릴 수 있도록 구현했습니다.
데이터 로딩이 완료되면 스켈레톤 UI가 실제 데이터로 대체되면서 화면에 자연스럽게 나타나도록 설계했습니다. 이를 통해 사용자들이 대기하는 동안에도 인터페이스의 구조를 시각적으로 이해할 수 있고, 데이터 로딩 과정을 매끄럽게 경험할 수 있는 웹사이트를 완성할 수 있었습니다.
